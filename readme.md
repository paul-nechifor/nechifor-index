# Nechifor Index

The index page of [my web site][ws].

![screenshot](screenshot.png)

## Usage

Get the submodule:

    git submodule init
    git submodule update

Install the dependencies:

    npm install

Build it with `intercessor` in `./build`:

    npm run build

Start it:

    npm run start

See it at [localhost:3000](http://localhost:3000).
